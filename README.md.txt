# Employee Attendance SQL Project

This project calculates the total work hours of employees based on their check-in and check-out times using SQL.

## Project Overview
- **Purpose**: Calculate total work hours, first check-in, last check-out, and the number of times an employee checks out in a day.
- **Technologies Used**: SQL.

## Files
- `employee_attendance.sql`: Contains the SQL code for the project.

## Example Output
| EmpID | Name  | First Check-In      | Last Check-Out       | Total Out Count | Total Work Hours |
|-------|-------|---------------------|----------------------|-----------------|------------------|
| 1     | Him   | 2024-01-03 10:08   | 2024-01-03 18:08    | 3               | 06:07           |
| 2     | Raj   | 2024-01-03 10:10   | 2024-01-03 15:12    | 2               | 04:49           |
| 3     | Anu   | 2024-01-03 10:12   | 2024-01-03 18:35    | 2               | 08:18           |

## How to Use
1. Copy the SQL code from `employee_attendance.sql`.
2. Execute it in your SQL environment.
3. Verify the results.
